# 레이어3(점수/알림/AI) 구체 기능 명세 + 유료 전환 트리거

레이어3는 **“불안(입소) 문제”를 해결하는 도구**이고, 커뮤니티에서 발생한 니즈를 흡수해 프리미엄으로 전환시키는 역할입니다.

---

## A. 입소 점수(Admission Score)
### 기능 범위(MVP)
- 입력: 아이 나이(반), 우선순위(맞벌이/형제/한부모/다자녀 등), 대기순번(선택)
- 출력:
  1) 점수(또는 등급) + 해설
  2) “지금 할 수 있는 액션(서류/대기전략)” 3개
  3) 유사 케이스(커뮤니티에서 익명 집계) 카드(후속)

### 데이터
- 초기에는 규칙 기반(정책/지자체 기준) + 운영 보정
- 장기적으로는 커뮤니티/TO 데이터를 통해 “실제 가능성” 추정 고도화

### 유료 제한(트리거)
- 무료: 월 1회
- Basic: 월 5회
- Premium: 무제한 + 결과 PDF/내보내기

**Paywall 트리거**
- 결과 화면에서 “다른 시설도 비교해보세요” 클릭 시
- 2번째 계산 시도 시(무료 한도 초과)

---

## B. TO 알림(충원/추가모집 감지 + 알림)
### 기능 범위(MVP)
- 구독: 관심 시설 1개(무료), 5개(Basic), 무제한(Premium)
- 알림 채널: 푸시(기본), SMS/이메일(옵션)
- 알림 히스토리: 최근 50개, 읽음 처리

### 감지 소스(단계적)
1) UJuz 커뮤니티 글/댓글에서 “TO/추가모집” 키워드 감지
2) 파트너 맘카페(외부)에서 합의된 피드/제보(= B2B2C 확장)
3) 공식 공지/웹(가능하면) – 단, 준수/허가 필수

### 유료 전환 트리거
- 무료로 1개 시설 알림 설정 후 “2번째 시설 추가” 클릭
- 알림을 2회 이상 오픈한 유저에게 “놓치지 않으려면” 메시지
- 알림 히스토리 ‘내보내기/공유’ 버튼(프리미엄)

---

## C. AI 봇(우쥬봇)
### 기능 범위(MVP)
- 질문 유형: 시설 정보/입소/비용/후기 요약/TO 알림 안내/비교 추천
- 컨텍스트: 시설ID, 아이 반, 대기순번, 우선순위, 위치(선택)
- 답변 길이: 300자 내 권장 + 다음 질문 제안

### 유료 제한(트리거)
- 무료: 일 5회
- Basic: 일 30회
- Premium: 무제한 + 우선 답변(속도) + 기록 검색

**Paywall 트리거**
- 6번째 질문 시도
- “결과를 표로 정리/저장” 같은 고가치 기능 클릭

---

## D. 전환 설계(가장 중요한 5개 트리거)
1) **성공 경험 후:** 점수 계산 완료 → “다른 시설도” → 결제
2) **불안 증폭 시점:** TO 알림 2번째 추가 → 결제
3) **반복 사용 시점:** AI 질문 6번째 → 결제
4) **가치 증명 후 공유:** 결과/히스토리 내보내기 → 결제
5) **커머스 브릿지:** 공구 참여/구매 경험 후 프리미엄 업셀

---

## E. 가격 패키징(권장)
- Basic: ₩9,900 (핵심 기능 확장)
- Premium: ₩29,900 (무제한 + 내보내기 + 우선지원)
- Family(선택): 프로필 2~3명/알림 공유(향후)

> 실제 운영에서는 “무료 → Basic” 전환이 핵심이고, Premium은 heavy user 위주로.
