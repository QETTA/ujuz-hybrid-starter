# 데이터/플랫폼 준수 원칙 (맘카페 파트너 모델 필수)

## 0) 결론
- **무단 수집(스크래핑) 금지**
- **운영자 합의 + 회원 고지 + 최소 수집 + 목적 제한**이 기본

---

## 1) 수집 허용 경로(권장)
1) 카페 운영자가 제공하는 공식 데이터 export(글 링크/메타)
2) 카페 내부 “TO 제보 폼/봇”을 통해 회원이 자발적으로 제출
3) 카페 운영자/모더레이터가 선별한 글만 공유

---

## 2) 최소 수집 원칙
- 개인 식별 정보(닉네임/전화/주소 등) 저장 금지
- 필요한 경우에도:
  - author는 hash 처리(복구 불가)
  - 원문 raw는 최소 기간 보관(예: 30일) 후 삭제

---

## 3) 목적 제한 / 2차 활용 금지
- 목적: TO 알림/입소 정보 품질 개선/카페 운영 리포트
- 광고 타게팅, 외부 판매 등은 금지(신뢰 핵심)

---

## 4) 투명성
- 카페 공지: “UJuz 엔진 연동 안내(무슨 데이터를, 왜 쓰는지)”
- UJuz 내: “출처 표시(파트너 카페) + 신뢰도 표기”

---

## 5) 분쟁 대응
- 요청 즉시 삭제/차단(카페/회원)
- 운영자 연락 채널 상시 제공
- 데이터 처리 기록(언제/무엇을/누가) 로그 유지
